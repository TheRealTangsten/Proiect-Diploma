Proiect realizat in IDE "PyCharm"
Rulat cu Python 3.9

Functii principale:
setup() - citeste si prepara toate datele. Obligatoriu de rulat de fiecare data.
euclidianDemo() - ruleaza creerea graficerlor si printarea performantelor pentru algoritmul Distanta Euclidiana Medie. Pentru descrierea argumentelor, a se consulta fisierul sursa ( contine comentarii )
pf.calcMeans() - calculeaza semnalele de referinta. Trebuie invocata inainte euclidianDemo() daca aceasta este folosita pentru date ne-preprocesate.
computeTimePreprocs() - afiseaza performante temporale pentru algoritmii de preprocesare
demoPreprocess() - afiseaza grafic un set de date in functie de continutul acestuia. A fost folosita strict experimental; pentru schimbarea setului de date se va decomenta linia respectiva din functie.
perceptron.runPerceptron() - ruleaza modelul perceptron. Configurabila; pentru mai multe detalii a se consulta definitia functii in fisierul sursa.
perceptron.runMLP() - ruleaza modelul MLP. Configurabila; pentru mai multe detalii a se consulta definitia functii in fisierul sursa.
CNN.runCNN() - ruleaza modelul CNN. Configurabila; pentru mai multe detalii a se consulta definitia functii in fisierul sursa.
